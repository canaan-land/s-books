# s-books

This repository contains detailed information about spiritual books, including:

- Book Titles
- Authors
- Publication Dates
- Tables of Contents
- Page Numbers

The aim is to systematically manage and reference book details.

## Structure

- `docs/`: Directory containing individual book details.
